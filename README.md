# DEV

1. Clone the repository.
2. Execute:
```
npm install
```
3. Install Nest CLI:
```
npm i -g @nestjs/cli
```
4. Run the db container: 
```
docker-compose up -d
```

## Stack:
* NestJS
* MongoDB
